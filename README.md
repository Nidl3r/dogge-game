## How to Use

1. **Install Python** (if you don't have it)
   - Download from https://python.org
   - During install, CHECK "Add Python to PATH"

2. **Install pygame**
   - Open Command Prompt
   - Run: `pip install pygame`

3. **Run the pet!**
   - Double-click `RUN_SCHNAUZER.bat`
   - Or run `python schnauzer_pet.py` from command line

Want to change the pet? Edit `schnauzer_pet.py`:
- Change colors at the top of the file
- Adjust `PET_SIZE` for bigger/smaller pet
- Modify `GROUND_Y` to change vertical position
- Add more tricks in the `tricks` list!
