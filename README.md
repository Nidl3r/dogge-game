## Troubleshooting

**Dog won't launch?**
- Make sure Python is installed
- Make sure you ran `pip install pygame`
- Try running from Command Prompt to see error messages

**Can't see the dog?**
- Check if it's on a different monitor
- Open zone editor to see where zones are placed
- Make sure zones aren't too small

**Dog disappeared?**
- It might have moved off-screen
- Close and relaunch
- Edit zones to keep it visible
